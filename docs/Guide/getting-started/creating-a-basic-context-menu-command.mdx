title: Creating a basic context menu command
---

Unlike creating the `Slash Command` without overriding the `Command` class, we need to override the
`registerApplicationCommands` always to create a `contextMenu` command. For example -

`Example`

```typescript ts2esm2cjs|{4-13}|{4-13}
import { Command } from '@sapphire/framework';

export class PingCommand extends Command {
  public constructor(context: Command.Context, options: Command.Options) {
    super(context, { ...options });
  }

  public override registerApplicationCommands(registry: ApplicationCommandRegistry) {
    registry.registerContextMenuCommand({
      name: 'ping',
      type: 'MESSAGE'
    });
  }
}
```

The above will create a `Context Menu` command called `ping` of `MESSAGE` type.

## Creating the `contextMenuRun` method

To handle the command callback when the context menu command is ran we need to override the `contextMenuRun` method of
command class.

```typescript ts2esm2cjs|{8-17}|{9-17}
import { Command } from '@sapphire/framework';
import type { Message } from 'discord.js';

export class PingCommand extends Command {
  public constructor(context: Command.Context, options: Command.Options) {
    // ...
  }

  public async contextMenuRun(interaction: Command.ChatInputInteraction) {
    return interaction.reply('Pong');
  }

  public override registerApplicationCommands(registry: ApplicationCommandRegistry) {
    // ...
  }
}
```
